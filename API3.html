Scripting API

Find here details of the scripting API that Generis provide.

This page is extensively furnished with example code that you are free to use for any purpose without crediting Generis.

Game Engine ¶

PROPERTIES ¶

All Scenes ¶

Usage: gGameEngine.scenes
Type: Dictionary
Keys: Scene names
Values: Scene Objects 
Description: Dictionary of all scenes created.

Active Scene ¶

Usage: gGameEngine.currScene
Type: String
Description: Name of currently active scene.

FUNCTIONS ¶

Create New Scene ¶

Usage: gGameEngine.newScene(name, canvas, width, height, x, y, camWidth, camHeight, camX, camY)
Description: Returns a Scene Object.
Parameter Type Description
name String The name of the scene to be created
canvas Canvas The canvas on which the scene will be shown
width Double Scene width
Height Double Scene height
x Double Scene x position relative to the canvas. Usually 0.
y Double Scene y position relative to the canvas. Usually 0.
cameraWidth Double Camera width
cameraHeight Double Camera height
cameraX Double Camera x position relative to the scene.
cameraY Double Camera y position relative to the scene. 
// Example
    var canvas = document.getElementById(canvas); 
    var scene = gGameEngine.newScene("CoolGame", canvas, 500, 500, 0, 0, 500, 500, 0, 0);

Close Scene ¶

Usage: gGameEngine.closeScene(name)
Description: Closes scene with given name.
Parameter Type Description
name String The name of the scene to be closed 
// Example
    gGameEngine.closeScene("CoolGame");

Start Scene ¶

Usage: gGameEngine.startScene(name)
Description: Starts scene with given name.
Parameter Type Description
name String The name of the scene to be started 
// Example

Delete Scene ¶

Usage: gGameEngine.delScene(name)
Description: Delete scene with given name.
Parameter Type Description
name String The name of the scene to be deleted 
// Example
    gGameEngine.delScene("CoolGame");

Create new Entities ¶

Usage: scene.spawnEntity(entityProps)
Description: Creates a new entity in the Scene

Entity Properties

{
    id: NAME,
    animbodies: ANIMBODIES,
    animFrames: ANIMFRAMES,
    initBody: INITBODY,
    zIndex: ZINDEX
}

Animation Bodies

{
    "animBody1": SETTINGS1,
    "animBody2": SETTINGS2,
        .
        .
        .
}

Settings

{
    pos: {
        x: X,
        y: Y
    },
    size: {
        width: WIDTH,
        height: HEIGHT
    },
    imageSizeFx: function(width, height) {
        .
        .
        .
        return [WIDTH, HEIGHT];
    },
    imageSizeFx: function(x, y) {
        .
        .
        .
        return [X, Y];
    },
    type: TYPE,
    categories: CATEGORIES,
    collidesWith: COLLIDESWITH,
    userData: USERDATA,
    enablePhysBody: ENABLEPHYSBODY,
    entityDef: ENTITYDEF,
    trueOffset: TRUEOFFSET,
    updBdByImgSize: UPDBDBYIMGSIZE,
    orientation: ORIENTATION,
    useCurrentPos: USECURRENTPOS,
    useImageSize: USEIMAGESIZE,
    showImage: SHOWIMAGE,
    useSprites: USESPRITES

}

Animation Frames

[
    ["animBody1", [ IMAGE1, IMAGE2, ... ]],
    ["animBody2", [ IMAGE1, IMAGE2, ... ]],
        .
        .
        .
]
Parameter Type Description
entityProps Dictionary All properties deffining the entity. It is formatted as shown above. 
// Minimum example
    var sets = { pos: { x: 0 , y: 500 } , size: { height: 100 , width: 1000 } };
    var charPs = { id: 'floor', animBodies: { Idle: sets }, animFrames: [[ "Idle", ["floor.png"]]], initBody: "Idle", zIndex: 1 };
    var entity = scene.spawnEntity(charPs);

Type: Integer
Description: Frame number of current Animation Body current frame.

Use Image Size ¶

Usage: entity.useImageSize
Type: bool
Description: If true, animation body frames will be rendered using their actual size. If false, animation body frames will be rendered using size provided in body settings.

Use Sprites ¶

Usage: entity.useSprites
Type: bool
Description: If true, animation body images should be in a spritesheet. If false, animation body images should use absolute paths.

Update Body By Image Size ¶

Usage: entity.updBdByImgSize
Type: bool
Description: If true, entity physics body will have same size as body image. If false, entity physics body will have a fix size.

Description: Offset image size by given amount.

Entity On Touch ¶

Usage: entity.addOnTouch(otherBody, point, impulse, physOwner)
Description: Specifies what happens when entity physics body gets into contact with any other entity physics body.
Parameter Type Description
otherBody Object Touched entity
point Vetor Body touched
impulse Vector Body touched
physOwner Object Physics Body of the Touched entity 
// Example
    entity.addOnTouch = function(otherBody, point, impulse, physOwner) { 
        entity.changeCurrentAnimation("animationClickDown"); 
        physOwner.changeCurrentAnimation("animationClickDown"); 
    }

Entity On Key Down ¶

Usage: entity.onKeyDown(event)
Description: Specifies what happens to the entity when keys on the keyboard are pressed.
Parameter Type Description
event Array Currently pressed keys.s 
// Example
    entity.onKeyDown = function(event){ 
        if( event[0] == "Left" ){
            var pos = this.physBody.GetPosition();
            pos.x -=  30;
            this.physBody.SetPosition(pos);
        }
    }
