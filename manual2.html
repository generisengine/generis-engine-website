        Getting Started ¶

        First Time Login ¶

        Generis stores all of your files in the cloud. Choose your preffered host, were your files will get stored.

        If this is the first time you login to the choosen host, then you will have a popup to allow Generis to Write/Read files from your host account.

        Here, we have choosen Google Drive as the others are not yet supported.

        A folder called “Generis Engine” will get automatically created in your host account. All of your files will get stored in here.

        At this point you may want to be in fullscreen. Click on the “FullScreen” icon on the bottom left corner.

        Enter the name of the game you want to builed, then click on “Create”.

        Files for your new game will then get created automatically for you.

        When this is done, you should have a screen which looks like this

        The window shown on the top left has one tab which, in this case, is a javascript editor. Please refer to the Code Editor section for more help.

        The window shown on the top right has one tab which, in this case, is the Game Viewer. Please refer to the Game Viewer section for more help.

        The window shown on the bottom has one tab which, in this case, is the Files Browser. Please refer to the Files Browser section for more help.

        To publish the game, go to Files - Publish.
